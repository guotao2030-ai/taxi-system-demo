# SwiftRide Taxi Dispatch System

SwiftRide is a responsive taxi dispatch system prototype built with HTML, CSS, and vanilla JavaScript. It provides passenger booking, driver order handling, operations monitoring, live vehicle status, and multilingual support in Chinese, English, and Saudi Arabic with RTL layout adaptation.

## Features

- Passenger booking with pickup, destination, car type selection, fare estimate, and instant ride request
- Driver workspace for accepting and completing orders
- Operations dashboard for monitoring orders, drivers, revenue, peak pricing, and capacity
- Map-style dispatch view with vehicle markers, route indicators, traffic index, and nearest driver status
- Multilingual UI: Chinese, English, and Arabic
- RTL support for Arabic layouts
- Responsive design for desktop and mobile screens
- Single-file deployment with no build step required
